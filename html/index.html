package.jsonを作成

npm init -y

gulpをインストール

npm install -D gulp --save-dev

必要なパッケージをインストール

npm install -D gulp-sass --save-dev

--save-devを付けるとpackage.jsonにインストールしたパッケージを記述してくれます。

  • gulp-sass（scssをcssに変換するパッケージ）
  • gulp-sass
  • aa
  • aa
  • aa
  • aa

パッケージをアンインストール

npm rm [パッケージ]

タスクの実行

npx gulp

タスクの終了

Ctrl + c
